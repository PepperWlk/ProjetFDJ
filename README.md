# Projet de jeu d’argent – FDJ (Unity 2D)

Ce dépôt contient un projet de jeu d’argent développé dans le cadre d’un cahier des charges défini par la FDJ (Française des Jeux). Le jeu est réalisé avec le moteur Unity en 2D et respecte un TDL (Travaux Dirigés Livrables) structurant l’ensemble du développement. Ce projet a été fait à l'occasion d'un stage de 2 mois au sein de la FDJ

## Objectif du projet

L’objectif principal est de concevoir un jeu d’argent en deux dimensions, conforme aux contraintes techniques et fonctionnelles imposées par la FDJ. Le projet met en œuvre des principes de game design, d’interface utilisateur ainsi que des mécanismes de jeu prenant en compte les aspects réglementaires et éthiques liés aux jeux d’argent.

## Technologies utilisées

* Unity (mode 2D)
* C#
* Visual Studio / Rider
* Git pour le versionnement

## Structure du projet

Le projet est organisé afin de refléter les différentes étapes du TDL, notamment :
* Analyse du besoin et spécifications fonctionnelles
* Conception du gameplay et des interfaces
* Développement sous Unity
* Tests et validation

Chaque partie du travail est identifiable dans l’organisation des dossiers et dans l’historique du dépôt.

## Installation et exécution

1. Cloner le dépôt :

   ```bash
   git clone <url-du-repo>
   ```

2. Ouvrir le projet dans Unity Hub.

3. Lancer la scène principale depuis le dossier `Assets/Scenes`.

4. Cliquer sur "Play" pour tester le jeu en mode éditeur.

## Remarques importantes

Ce projet s’inscrit dans un cadre pédagogique et expérimental. Il ne constitue pas une application de jeu d’argent officiellement exploitée par la FDJ.
